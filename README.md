# TaskMaster (Flask TODO Web App)

A task manager built with Flask + SQLAlchemy. Users are able to create, update, and delete tasks with persistent storage.

# Demo
Not hosted yet. Run locally using the steps below.

# Features
- Create tasks
- View tasks ordered by creation time
- Update existing tasks
- Delete tasks
- Persistent storage (allows users to come back)

# Tech Stack
Backend: Python, Flask
Database: SQLAlchemy ORM, SQLite (local)
Frontend: HTML/CSS + Jinja templates

# Getting Started (Run Locally)
1. Need to have Python 3.9+ installed
2. Clone the repo
3. git clone https://github.com/Hj4704/TaskMaster-Project.git
4. cd <TaskMaster-Project>
5. python3 -m venv .venv
6. source .venv/bin/activate
7. pip install -r requirements.txt
8. python myapp.py
9. Open in browser go to: http://127.0.0.1:5000
